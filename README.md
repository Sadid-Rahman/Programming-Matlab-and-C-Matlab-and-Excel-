# 2D Battleships & Mini-Games
MATLAB Game Development with CLI and AI Opponents

# Project Description 
This project includes a 2D Battleships game and other mini-games built in MATLAB. 
The games feature an interactive GUI, player vs AI gameplay, and scoring systems.

# Features
- Turn-based gameplay for player vs AI
- Scoring system and input validation
- Object-oriented programming for game entities
- Additional mini-games demonstrating programming concepts

# Technologies 
- MATLAB
- Object-Oriented Programming
- GUI Design
